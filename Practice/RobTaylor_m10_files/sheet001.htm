To be honest, I was surprised to find any styling on this website. But there it was, Comic Sans Serif Font. (and some others, but not in the <style> section).

So, it took me a while to figure it out, but there is a space after font-family and before the colon. To remove this warning, you just remove that space.

Update elements using CSS:
 . font face => class { font-family: 'Comic Sans Serif' }
 . img border  => img { border: none }
 . width  => img { width: 100% }
 . cellpadding, cellspacing  => table { border-spacing: 2px }
 . valign  => .container { display: flex; align-items: center }
