Taylor's History with Websites and HTML

1996

Taylor sees his first website using Mozilla. It had text and Michael Jackson's Black or White face morphing image from 1991 on it:

1996

Taylor builds his first website using just HTML, images, and music.

1997

Taylor uses ColdFusion to build a website to create sales bids to compete against Wal-Mart and other box stores.

2000

Taylor is introduced to XML and how it could be used in place of fixed file formats for transmitting data. Adding another Markup language to his repertoire.

2001-2023

The world of the web moves on while Taylor was programming and doing other things...

2024

Taylor is laid off from his job and enrolls at Bellevue College to sharpen his skills. The result with HTML, CSS, and JavaScript has been great. The sky is the limit using just these tools. I know there are other websites out there that can accelerate the possibilities even more. But learning how these are tied together has been refreshing.
