Welcome to Taylor Made

This is a website that is applying several principles learned in DEV 109, Introduction to Web Development. I took this course in the Fall of 2024 at Bellevue College. The goal of this site is to try to use several techniques with HTML5 (including form data), CSS, and JavaScript.

My goal is to have a simple logo with links in the header of every page, like a menu. Every page will have a footer that includes copyright information and CSS/HTML Validators and a link to "Grow Deep," my "Creative" page. The links (menus) will navigate to new page for each menu choice.

Select a page from the menu to get started.
